Gizlilik Politikasi

Son guncelleme: 27 Nisan 2026

Ozet: QuickTranslate, gizliliginizi ciddiye alir. Kisisel verilerinizi toplamaz, satmaz veya ucuncu taraflarla paylasmaz. Ceviri istekleriniz yalnizca ceviri hizmetini saglamak amaciyla islenir.

1. Giris

Bu Gizlilik Politikasi, QuickTranslate Chrome tarayici eklentisi ("Eklenti", "biz", "bizim") tarafindan kullanici verilerinin nasil toplandigi, kullanildigi ve korundugunu aciklar. Eklentiyi kullanarak bu politikayi kabul etmis sayilirsiniz.

QuickTranslate, SlimPDF ("Sirket") tarafindan gelistirilmis ve yonetilmektedir.

2. Toplanan Veriler

2.1 Ceviri Verileri

Ceviri islemi sirasinda sectiginiz metin, ceviri hizmetini saglamak amaciyla ucuncu taraf ceviri API'sine (MyMemory) gonderilir. Bu veri:

  • Yalnizca ceviri amaciyla kullanilir
  • Sunucularimizda saklanmaz
  • Ceviri islemi tamamlandiktan sonra iletim sona erer

2.2 Yerel Depolama

Asagidaki veriler yalnizca tarayicinizin yerel depolama alaninda (chrome.storage.local) saklanir ve hicbir sunucuya gonderilmez:

  • Ayarlar: Dil tercihleri, tema secimi, sesli okuma ve gecmis tercihleriniz
  • Ceviri Gecmisi: Son 100 ceviriniz (istege bagli, kapatilabilir)
  • Kelime Haznesi: Kaydedilen kelimeler, es anlamlilari, kelime turleri ve SRS tekrar zamanlari
  • Fosforlu Kalem: Web sayfalarinda isaretlediginiz metinler, URL bazli olarak yerel depoda saklanir
  • Istatistikler: Toplam ceviri sayisi, dil kullanim istatistikleri ve haftalik ilerleme verileri
  • API Anahtarlari: Girdiginiz OpenAI ve DeepL API anahtarlari sifrelenmis olarak yerel depoda tutulur
  • YouTube Tercihleri: Dual Subtitle kutusunun konumu, yazı boyutu ve renk tercihleri yerel olarak saklanır

Onemli: Bu verilerin tamami cihazinizda yerel olarak saklanir. Sunucularimiza veya herhangi bir ucuncu tarafa aktarilmaz.

2.3 Toplanmayan Veriler

QuickTranslate asagidaki verileri toplamaz:

  • Kisisel kimlik bilgileri (ad, e-posta, telefon)
  • Konum verileri
  • Tarama gecmisi
  • Cerezler (cookies)
  • Finansal bilgiler
  • Saglik verileri
  • Kimlik dogrulama bilgileri

3. Ucuncu Taraf Hizmetleri ve API'ler

Eklentimiz, size en iyi hizmeti sunmak icin cesitli guvenilir ucuncu taraf API'leri kullanir. Bu servislere sadece ilgili ozelligi kullandiginizda veri gonderilir:

3.1 Ceviri Hizmetleri (MyMemory & DeepL)

Varsayilan ceviriler MyMemory API araciligiyla yapilir. Eger ayarlardan kendi DeepL API anahtarinizi girerseniz, ceviriler dogrudan DeepL'e gonderilir. API anahtariniz sadece tarayicinizda sifreli saklanir ve asla bizim sunucularimiza gonderilmez.

3.2 Yapay Zeka Ozeti (OpenAI)

Eger "Ozetle (AI)" ozelligini kullanmak icin kendi OpenAI API anahtarinizi girerseniz, sectiginiz metinler ozetleme amaciyla OpenAI sunucularina iletilir. API anahtariniz yerel olarak saklanir.

3.3 PDF ve Gorsel Ceviri (OCR.space)

Gorsel veya PDF uzerinden metin kirparak ceviri yaptiginizda, goruntu metne donusturulmek uzere OCR.space API servisine gonderilir. Gorseller islem sonrasi sunucularindan aninda silinir.

3.4 Kelime Baglami ve Gorseller (Dictionary API & Unsplash)

Kelime haznesindeki Ingilizce kelimelerin ornek cumleleri, es anlamlilari ve kelime turleri Free Dictionary API (dictionaryapi.dev) uzerinden anlik olarak cekilir. Flashcard modundaki gorsel hafiza kartlari icin Unsplash uzerinden anonim gorsel istekleri yapilir.

3.5 Telaffuz Denetimi (Speech Recognition)

Flashcard modundaki "Telaffuz Pratigi" ozelligi, tarayicinizin yerlesik Web Speech API altyapisini kullanir. Ses veriniz yerel olarak veya tarayicinizin varsayilan hizmet saglayicisi tarafindan islenir, tarafimizca kaydedilmez veya toplanmaz.

3.6 Fosforlu Kalem ve Okuma Modu Verileri

Web sayfalarinda isaretlediginiz fosforlu kalem verileri yalnizca yerel tarayici deponuzda (chrome.storage.local) URL bazli olarak saklanir. Bu veriler hicbir sunucuya gonderilmez. Okuma Modu ise sayfa icerigini yalnizca tarayicinizda yeniden duzenler, herhangi bir veri aktarimi yapilmaz.

3.7 YouTube Altyazı Verileri

YouTube Altyazı Çevirmeni özelliği, yalnızca videoda o an oynatılan altyazı segmentlerini yakalar ve seçtiğiniz API (MyMemory/DeepL/OpenAI) aracılığıyla çevirir. Bu veriler:

  • Videonun asıl içeriğine veya YouTube hesabınıza müdahale etmez
  • Çevrilen altyazı metinleri hiçbir sunucuda saklanmaz
  • Sadece anlık görüntüleme amacıyla kullanılır

4. Izinler

Eklenti asagidaki Chrome izinlerini kullanir:

  • activeTab: Aktif sekmede metin secimini algilamak icin
  • storage: Ayarlari, kelimeleri, fosforlu kalem verilerini ve ceviri gecmisini yerel olarak saklamak icin
  • contextMenus: Sag tik menusunde ceviri, fosforlu kalem ve okuma modu secenekleri sunmak icin
  • tts: Sesli okuma ozelligini saglamak icin
  • alarms & notifications: Kelime tekrarlari (SRS), Pomodoro zamanlayicisi ve haftalik basari raporu bildirimlerini gostermek icin

5. Veri Guvenligi

Verilerinizin guvenligini saglamak icin asagidaki onlemleri aliyoruz:

  • Tum API iletisimleri HTTPS uzerinden sifrelenerek yapilir
  • Hicbir kisisel veri sunucularimizda saklanmaz
  • Yerel veriler yalnizca tarayicinizin guvenli depolama alaninda tutulur
  • Verileriniz ucuncu taraflara satilmaz veya paylasilmaz

6. Kullanici Haklari

GDPR ve KVKK kapsamindaki haklariniz:

  • Erisim Hakki: Hakkinizda tutulan verilere erisim isteyebilirsiniz
  • Silme Hakki: Ceviri gecmisinizi eklenti ayarlarindan silebilirsiniz
  • Itiraz Hakki: Veri isleme faaliyetlerimize itiraz edebilirsiniz
  • Tasinabilirlik: Ceviri gecmisinizi CSV, PDF veya Anki uyumlu formatta disa aktarabilirsiniz

7. Cocuklarin Gizliligi
